## Development Workflow
1. **Edit** in VS Code (fast, responsive)
2. **Build** in Visual Studio 2022 (proper .NET SDK)
3. **Commit** via GitHub Desktop (version control)

## Data Architecture
- **Drawers**: Thematic quest groupings (MSQ, Features, etc.)
- **Bands**: Expansion-level organization
- **Patches**: Chronological release groupings

## Key Design Decisions
- JSON for quest data (human-readable, version-controllable)
- Service pattern for data access
- ImGui for UI (Dalamud standard)
